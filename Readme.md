Seamless Ads (in-movie clickable ads)

Run sample scenes with Kroger API results:

python -m seamless_ads seamless_ads/samples/scene_1.json seamless_ads/samples/user_1.json
python -m seamless_ads seamless_ads/samples/scene_2.json seamless_ads/samples/user_1.json

Requires Kroger API credentials:
KROGER_CLIENT_ID, KROGER_CLIENT_SECRET, (optional) KROGER_ACCESS_TOKEN
